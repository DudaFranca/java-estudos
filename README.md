# Java Estudos — Fase 1

Repositório de exercícios práticos desenvolvidos durante o aprendizado de Java, partindo do zero com base em conhecimento prévio de PHP.

## Conteúdo

### Métodos e Strings

Criação de uma classe `ManipuladorTexto` que recebe uma String e exibe informações como tamanho, versão em maiúsculo/minúsculo, verificação de conteúdo e remoção de espaços.

### ArrayList

Criação de uma classe `GerenciadorNomes` com métodos para adicionar, remover, verificar e listar nomes armazenados em um ArrayList.

### HashMap

Criação de uma classe `Estoque` que gerencia produtos e quantidades usando HashMap, com métodos para adicionar, atualizar, remover e listar produtos.

### Scanner

Evolução do GerenciadorNomes com interface interativa via terminal, usando Scanner e menu com switch para o usuário interagir com a lista em tempo real.

## Tecnologias

- Java
- IntelliJ IDEA
