# ☕ Java Studies

Repositório criado para documentar minha jornada de aprendizado em **Java**, reunindo exercícios práticos desenvolvidos durante os estudos.

O objetivo deste projeto é praticar os principais conceitos da linguagem, evoluindo gradualmente do básico até temas mais avançados.

---

## 📚 Conteúdo

### 📌 Fase 1 — Fundamentos

| Tema                  | Exercício                                                                                                                                                                                                                                                                      |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Métodos e Strings** | Criar uma classe `ManipuladorTexto` contendo um método `analisarTexto`, capaz de receber uma `String` e exibir a quantidade de caracteres, converter o texto para maiúsculas e minúsculas, verificar a existência de uma palavra e remover espaços em branco nas extremidades. |
| **ArrayList**         | Desenvolver uma classe `GerenciadorNomes` utilizando `ArrayList`, implementando funcionalidades para adicionar, remover, verificar a existência e listar nomes armazenados na coleção.                                                                                         |
| **HashMap**           | Criar uma classe `Estoque` utilizando `HashMap` para armazenar produtos e suas quantidades, permitindo adicionar, atualizar, remover e listar os itens cadastrados.                                                                                                            |
| **Scanner**           | Evoluir a classe `GerenciadorNomes`, adicionando uma interface interativa no terminal com `Scanner` e um menu baseado em `switch` para que o usuário possa executar as operações da lista em tempo real.                                                                       |

## 🚀 Fase 2 - POO

| Tema                 | Exercício                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Encapsulamento 1** | Crie uma classe `ContaBancaria` com os atributos privados `titular`, `saldo` e `numeroConta`. Ela deve ter um `construtor` que recebe titular e numeroConta, iniciando o saldo em zero. E deve ter `métodos` para depositar um valor, sacar um valor validando se há saldo suficiente, e exibir os dados da conta. No `main`, crie duas contas e faça operações nas duas.                                                                                                                                                                     |
| **Encapsulamento 2** | Crie uma `classe Produto` com os atributos privados `nome`, `preco` e `quantidadeEmEstoque`. O `construtor` deve receber os três atributos. A classe deve ter os seguintes métodos: `aplicarDesconto` que recebe um percentual e reduz o preço, mas só aplica se o percentual for entre 0 e 100; `adicionarEstoque` que recebe uma quantidade e soma ao estoque; `vender` que recebe uma quantidade e subtrai do estoque, mas só permite a venda se houver quantidade suficiente; e `exibirDados` que mostra todas as informações do produto. |
